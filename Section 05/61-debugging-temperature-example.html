61 - Debugging with the Console and Breakpoints

Example of debugging from the console. In this example also contains creating an Object inside of a function also converting a string into a Number.


const measureKelvin = function () {
const measurement = {
type: 'temp',
unit: 'celsius',

FIX
value: Number(prompt('Degrees celsius')),
};

FIND
console.table(measurement);

const kelvin = measurement.value + 273;
return kelvin;
};

IDENTIFY
console.log(measureKelvin());
