Conditional (Ternary) Operator

A conditional is for example an if statement or a switch statement, but there is also a conditional statement. It is like an if statement but you can write it all in one line.

const age = 25;

age >= 21 ? console.log("I want a big beer üç∫") : console.log("Boo, give me a soda ü•§");

? is like the if
: is like the else
Example: Storing a coditional operator in a variable.

const drink = age >= 18 ? "Beer!" : "Water";
console.log(drink);

Answer: Beer!
Example: Using a conditional operator in a template literal.
console.log(`I like to drink ${age >= 18 ? "Beer!" : "Water"}`);
