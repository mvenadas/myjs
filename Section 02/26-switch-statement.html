26 - Switch Statement

Alternative way of writing an if statement. Lesser used but should know. Pros: little cleaner code.

const day = "friday";

switch (day) {
case "monday":
console.log("Eat tacos!");
break;
case "tuesday":
console.log("Eat pizza");
break;
case "wednesday":
console.log("Eat spagetti");
break;
case "thursday":
console.log("Eat burgers");
break;
case "friday":
console.log("Eat chili fries");
break;
case "saturday":
console.log("Eat soup");
break;
case "sunday":
console.log("Eat steak");
break; }

Answer: Eat chili fries
