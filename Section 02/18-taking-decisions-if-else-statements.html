18 - Taking Decisions - if/else Statements

if/else statements are identified as a controlled structure.

A if/else contains a condition is a block of code that returns a "true" or "false" answer.

Exercise: Is Colin old enough to drive. If not, how many years until he is?

const age = 15;

if (age >= 18) {
console.log("Colin can start driving!");
} else {
const yearsLeft = 18 - age;

console.log(`Colin is too young. Wait another ${yearsLeft} years!`); }

Answer: Colin is too young. Wait another 3 years!

What century were you born in?

const birthYear = 1979;

let century;
if (birthYear <= 2000) {
century = 20 + `th`;
} else {
century = 21 + `st`;
}

console.log(`You were born in the ${century} century!`);

Answer: You were born in the 20th century!
