Equality Operators

Examples of equality operators

The Strict Equality Operator === compares both sides of the center equal sign to see of both values are equal to each other. It only returns true with both values are the same.

const age = 18;
if (age === 18)
console.log("You just became an adult!");

Note: If the if statement is exact it does not require the curly brackets {} to execute any code.

Examples:

18 === 18
Answer: True

18 === 19
Answer: False

The Loose Equality Operator == will perform Type Coercion. In the example below, == will convert the "18" string to a number.

Note: This can cause issues in your code, so try yo avoid using this operator as much as you can.

Example:

"18" == 18
Answer: True

Example using the prompt function with the Strict Equality Operator ===

Exercise: Use the prompt function. Enter the number 15 or 30 in th prompt box. In your code, manually convert the "favourite" variable to a number. Then use the if / elseif / else statement.

Note: After entering the number in the promt box, JS will identify the value as a string. Since we will be using the === operator you will need to manually convert the string to a number.

const favourite = Number(prompt("What is your favourite number"));
console.log(favourite);

if (favourite === 15) {
console.log("Number 15, YAY!!!");
} else if (favourite === 30) {
console.log("Number 30, YAY!!!");
} else {
console.log("Sorry, please input the number 15 or 30!");
}

Example using the Strict Inequality Operator !==

if (favourite !== 15) {
console.log("Wait, what about all the other numbers?");
