Strings And Template Literals

Practice concatenating strings.

Defined Variables:
const firstName = "Michael";
const job = "Front End Developer";
const birthYear = 1979;
const year = 2023;

const michael = "I'm " + firstName + ', a ' + (year - birthYear) + ' ' + 'years old' + job + '!';

Answer: I'm Michael, a 44 years old Front End Developer!

Starting with ES6 - template literals were introduced.

const michaelNew = `I'm ${firstName}, a ${year - birthYear} years old ${job}!`;

Answer: I'm Michael, a 44 years old Front End Developer!

HTML can even be used within the template literals.
