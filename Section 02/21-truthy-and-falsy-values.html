21 - Truthy and Falsy Values

Falsy: Are values that are not exactly false but they will become false when we try to convert them to a boolean. All of these 5 values will be converted to false when we try to convert as a boolean.

  1. 0 (Zero)
  2. Empty String
  3. Undefined
  4. Null
  5. NaN

console.log(Boolean(0));
Answer: False

console.log(Boolean(undefined));
Answer: False

console.log(Boolean("Michael"));
Answer: True

console.log(Boolean({}));
Answer: true

console.log(Boolean(""));
Answer: False

Example of an if/else statement where Javascript converts zero to false and console logs the else statement.

const money = 0;
if(money) {
console.log("Invest it");
} else {
console.log("Get a job!");
}

Answer: Get a job!

Truthy: Any string or number that is not zero or part of the list above will be converted to true.
