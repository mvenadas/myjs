20 - Type Conversion and Coercion

Converting between types is essential in all programming languages. Example: Converting a string to a number or a string to a number.

  • Type Conversion: When we manually convert from one type to the other.
  • Coercion: When Javascript automatically converting types behind the scences.

Example of conversion manually

How to manually convert a string to a number:

const inputYear = "1991";

console.log(Number(inputYear) + 18);
Answer: 2009

How to manually convert a number to a string:
console.log(String(inputYear) + 18);
Answer: 199118

Note: Number() and String() are preset functions in Javascript.

Example of Coercion
Whenever an operator is dealing with two values with different types. Js will then convert on of the values to match the other value.

console.log('I am ' + 23 + ' years old ');
Answer: I am 23 years old

console.log("23" - "10" - 3);
Answer: 10
Note: Javascript actually converted the strings into a number. The minus sign triggered the conversion. You can also use the "*" and "/" operators and it will convert to a number.

console.log("23" + "10" + 3);
Answer: 23103
Note: Javascript converted it all to a string.
