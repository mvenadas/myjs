Data Types

The 7 Primative Data Types

  1. Number: Floating point numbers = Used for decimals and integers
    Example: let age = 23;
  2. String: Sequence of characters = Used for text
    Example: let firstName = 'Michael';
  3. Boolean: Logical type that can only be true of false = Used for taking decisions
    Example: let fullAge = true;
  4. Undefined: Value taken by a variable that is not yet defined ("empty value")
    Example: let children;
  5. Null: Also means "empty value"
  6. Symbol (ES2015): Value that is unique and cannot be changed
  7. BigInt: Larger integers that the Number type can hold
