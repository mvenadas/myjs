Object: Dot Vs. Bracket Notation

const michael = {
firstName: "Michael",
lastName: "Venadas",
age: 2023 - 1979,
job: "Front End Developer",
friends: ["Richard", "Mauro", "Josh"],
};

Ways to call out the obect properties and its data
console.log(michael.lastName);
console.log(michael["lastName"]);

Using the bracket notation method using an expression. Note: You cannot do this using the dot method
const nameKey = "Name";
console.log(michael["first" + nameKey]);
console.log(michael["last" + nameKey]);

Using the prompt() to accept an input or prompt an error message
const interestedIn = prompt(
"What do you want to know about Michael? Choose between firstName, lastName, age, job, and friends"
);

if (michael[interestedIn]) {
console.log(michael[interestedIn]);
} else {
console.log(
"Wrong request! Choose between firstName, lastName, age, job, and friends"
);
}

Adding new properties to the object by using dot or bracet notations
michael.location = "Budapest";
console.log(michael);
