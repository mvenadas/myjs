Object Methods

Functions can be added to objects. Any function that is attached to an object is called a Method.

Example using the "this" keyword. Using this will elimate having to pass through the arguement.
Since the object "michael" contains the function "calcage" using "this" will point to that property.

const michael = {
firstName: "Michael",
lastName: "Venadas",
birthYear: 1979,
job: "Front End Developer",
friends: ["Richard", "Mauro", "Josh"],
hasDriversLicense: true,
calcAge: function (birthYear) {
return 2023 - this.birthYear;
},

getSummary: function () {
return `${this.firstName} is a ${this.calcAge()}-year old ${michael.job }, and has ${this.hasDriversLicense ? "a" : "no"} driver's license.`;},
};
