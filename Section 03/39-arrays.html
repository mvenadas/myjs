39 - Arrays

Arrays are a data structure. Two most important data structures are arrays and objects.

Note: You cannot use a statement in the brackets of an Array when calling out. You can use an expression.

The literal syntax of creating an Array
const friends = ["Richard", "Mauro", "Josh"];
console.log(friends);

A different way of creating an Array, but with a function
const years = new Array(1991, 1984, 2008, 2020);
console.log(years);

Gets the number of elements in the Array
console.log(friends.length);

Gets the last element of the Array. Note: Since the Array is "0" based you will need to use the "- 1" within the call out
console.log(friends[friends.length - 1]);

Changing an element to the Array
friends[2] = "Alex";
console.log(friends);

Holding differents types of information
const lastName = "Venadas";
const michael = [ "Michael", lastName, 2023 - 1979, "Front End Developer", friends, ];

console.log(michael);
