Iteration - The For Loop

Loop is a control structure. Allows us to automate repetitive tasks. The statement loop has 3 parts. The For loop will keep running while the condition is true.

  1. Initial value of the counter Ex: 1
  2. Logical condiion that is evaluated before each iteration of the loop
  3. Increase the amount of the loop
for (let rep = 1; rep <= 10; rep++) {
console.log(`Lifting weights repetition ${rep}`);
}
