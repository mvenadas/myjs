Basic Array Operations (Methods)

There are numerous methods that can be used with an Array.

const friends = ["Richard", "Mauro", "Josh"];

Push - will add an element to the end of the array. Note that Push is a function
friends.push("Alex");
console.log(friends);

Unshift - Adds an element to the begining of the array
friends.unshift("Jordan");
console.log(friends);

Pop - Removes the last element of the array
friends.pop();
console.log(friends);

Shift - Removes the first element of the array
friends.shift();
console.log(friends);

IndexOf - Returns the number value (the place) of the element
console.log(friends.indexOf("Mauro"));

Includes - Will return True or False of the element exists in the array
console.log(friends.includes("Josh"));
console.log(friends.includes("Michael"));
